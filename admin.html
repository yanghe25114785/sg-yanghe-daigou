🔧 初始配置

只需设置一次，完成后所有商品操作都会自动保存到云端，无需手动复制代码。

1
注册 JSONbin.io（免费存储数据）
用于永久保存你的商品数据。注册后：
① 点左侧 Bins → Create Bin
② 内容填 {"goods":[]}，点 Create
③ 复制地址栏里的 Bin ID（最后一串字符）
④ 点右上角头像 → API Keys → 复制 Secret Key
打开 jsonbin.io ↗
2
注册 ImgBB（免费图片存储）
用于上传商品图片。注册后：
① 点右上角头像 → API
② 复制你的 API Key
打开 imgbb.com ↗
