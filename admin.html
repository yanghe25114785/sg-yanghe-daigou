🔧 初始配置

只需设置一次，完成后所有操作自动保存到云端。

1
注册 JSONbin.io（免费存储数据）
注册后：① Bins → Create Bin，内容填 {"goods":[]} → Create
② 复制地址栏 Bin ID（最后一串字符）
③ 头像 → API Keys → 复制 Secret Key
打开 jsonbin.io ↗
2
注册 ImgBB（免费图片存储）
注册后：头像 → API → 复制 API Key
打开 imgbb.com ↗
